# HealthCare+ Web Projects

This repository contains two mini web-based projects developed as part of my 2nd Semester coursework. Both projects are built using basic web technologies like HTML, CSS, and JavaScript, focusing on real-world functionality and simple user interaction.

---

## Project 1: Doctor Appointment System

The Doctor Appointment System is a simple web application that allows users to book doctor appointments easily. The goal of this project is to simulate a basic hospital booking system with multiple pages and user interaction.

### Features:

* Book appointments by selecting doctor, date, and patient details
* Payment page simulation with validation
* Appointment confirmation page
* View all booked appointments
* Reset all appointments option
* Input validation (date check, text-only fields, numeric inputs)
* Local storage used to store appointment data

### Technologies Used:

* HTML
* CSS
* JavaScript

---

##  Project 2: Health Calculator (BMI + Calorie Calculator)

This project is a Health Calculator that helps users calculate their Body Mass Index (BMI) and daily calorie requirements. It also provides a basic diet suggestion based on the calculated BMI.

### Features:

* BMI calculation
* Daily calorie requirement calculation
* Diet plan suggestion based on BMI category
* Activity level selection
* Logs system to store previous calculations
* Clear logs functionality
* Input validation (numbers only fields)
* Multi-page interface using JavaScript

### Technologies Used:

* HTML
* CSS
* JavaScript

---

##  Key Concepts Used

* DOM Manipulation
* Event Handling
* Form Validation
* Local Storage
* Multi-page UI using JavaScript
* Basic responsive layout

##  Purpose of the Projects

These projects were created to practice front-end web development concepts and understand how real-world applications like booking systems and health tools work. The focus was on building functional and user-friendly interfaces rather than using advanced frameworks.

##  How to Run

1. Download or clone the repository
2. Open the project folder
3. Open the `.html` file in any browser

No installation or server is required.
